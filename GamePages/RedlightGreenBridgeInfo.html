Project Overview

Red Light Green Bridge is a game I developed in Unreal Engine for a game jam where the theme was "never give up". The game is designed to be challenging and requires players to cross the bridge while avoiding the red lights.

Key Features

  • Unique mechanics that require players to think strategically

Development Insights

I developed this game in Unreal Engine for a game jam where the theme was "never give up". The goal was to create a challenging game that would test players' skills and perseverance. I used Unreal Engine's Blueprints system to create the game's mechanics, which allowed me to quickly prototype and iterate on ideas. The game is designed to be difficult, but also rewarding, as players must learn from their mistakes and improve their skills to succeed.
