Poseidon's Wrath
A Deep Dive

My first game I developed with my team while in college using Unity.

Project Overview

Poseidon's Wrath is a fast-paced action game where players must defeat waves of enemies using a a variety of weapon powerups. You play as Poseidon, the god of the sea, and you must defeat all the enemies that come your way while surviving as long as possible.

Key Features

  • Arcade like infinite shooter
  • Never ending gameplay experience
  • Variety of weapon powerups
  • Unique art style

Development Details

  • Engine: Unity
  • Language: C#
  • Status: Completed
  • Platforms: Windows

Development Insights

This game was my first project I worked on with my team while in college. I assisted by creating the UI, game manager, and created the music and audio from scratch. This project taught me how to use version control with GitHub, how to work in a team environment, and how to create a game from concept to completion.
