The Village of Sacrifice
A Deep Dive

A 2D platformer I made in Unity for a game jam where the theme was "sacrifice".
You play as a person who is trapped in a loop of sacrificing villagers to escape a nightmare.

Project Overview

You find yourself trapped in a loop. A nightmare. A recursion hell some would call it. You made the worst deal with the devil and now you have to sacrifice all the villagers in town! Surely once you finish committing this heinous crime you will be set free...Right?

Key Features

  • Unique sacrifice mechanic that drives the gameplay
  • 2D retro aesthetic reminiscent of classic platformers

Development Details

  • Engine: Unity
  • Language: C#
  • Status: Completed
  • Platforms: Windows

Development Insights

This game was developed during a game jam, where I contributed to the design and implementation of the sacrifice mechanic. The project was a collaborative effort, and we aimed to create a unique gameplay experience that revolves around the theme of sacrifice. The game features a dark narrative and challenging platforming elements, making it a memorable entry in the game jam.
