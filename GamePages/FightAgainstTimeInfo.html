Fight Against Time
A Deep Dive

A small game jam game I worked on where you have to defeat all
the enemies using time manipulation!

Project Overview

The king has ordered you to take out the enemies, and he wants it done ASAP. Luckily, not only do you know how to handle a sword, but you can also control time!

Key Features

  • Unique time manipulation mechanics
  • Challenging game with replayability
  • Wave like spawn system

Development Details

  • Engine: Unity
  • Language: C#
  • Status: Completed
  • Platforms: Windows

Development Insights

I created this project for a game jam, where the theme was action. I was curious with time manipulation, and I wanted to learn how to do a dodge "witch time" effect . This project taught me how to effectivly manage the time scale in unity as well as how to create a game from concept to completion in a small time frame.
