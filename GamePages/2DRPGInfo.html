Project Overview

A bullet hell game I made in Unity for a college project. The player controls a character that must navigate through waves of enemies and obstacles, all while trying to survive as long as possible.

Key Features

  • Unique bullet hell mechanics
  • Never ending gameplay experience
  • Mouse only controls

Development Insights

I created this project while in collge as a fun way to learn more about Unity and C#. I purposely made the game a bullet hell to challenge myself with the mechanics and to create a unique gameplay experience. The project was well received by my peers and instructors, and it helped me solidify my understanding of Unity's physics and input systems.
