Insomnia
A Deep Dive

A horror game currently in progress developed in Unity.
You play as a boy named Timmy who needs to conquer his demons and try to fall asleep.
Hide under beds, in closets, whatever it takes to sleep in this first person horror game!

Project Overview

Insomnia is a first-person horror game where you must help Timmy survive the night by avoiding monsters and managing his fear. The game features atmospheric environments, stealth mechanics, and a narrative-driven experience.

Key Features

  • Immersive horror atmosphere with dynamic lighting and sound
  • Stealth gameplay: hide under beds, in closets, and avoid detection
  • Fear management system: Timmy's fear affects gameplay and visuals
  • Multiple endings based on player choices
  • Developed in Unity using C#

Development Details

  • Engine: Unity
  • Language: C#
  • Status: In Progress (Alpha available)
  • Platforms: Windows PC

Development Insights

This project challenged me to design AI behaviors for monsters, implement a fear system, and create immersive audio/visual effects. I learned a lot about Unity's lighting and animation systems, as well as best practices for organizing a larger codebase.
