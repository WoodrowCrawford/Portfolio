Project Overview

Insomnia is a first-person horror game I made by myself in Unity (C#). It started as my entry for a 48-hour game jam, but I kept going with it, and it's become the longest project I've worked on so far. Over a year of development across 36 releases.

Welcome to Insomnia, the state of mind in which dreams become stagnant, and nightmares run rampant. But in this nightmarish void, nothing is as it seems. Labeled as insane and dismissed as hallucinations, you hear the shadows whispering truths. As the darkness deepens, the monsters in your mind awaken. Can you escape this haunting abyss, or will you be another lost soul in the endless night?

Key Features

  • Immersive horror atmosphere with dynamic lighting and sound
  • Players can interact with the environment to hide from or distract monsters
  • 3D spatial audio with ambient occlusion (best played with headphones)
  • Cinemachine and Timeline used for scripted scares and pacing
  • Customizable settings

What I worked on

This started as a 48 hour game jam, but I liked it too much to leave it there, so I went back and rebuilt it into actual systems instead of one giant messy script. Most of my time went into the enemy AI and the systems that hold the whole game together. The main things I built:

  • Five enemies, each built with the finite state machine method, using Unity's NavMesh for movement
  • An event system so the different parts of the game can talk without being tangled together
  • A shared interaction setup the bed, doors, light switches, and toys all run off
  • The sleep mechanic the whole game is based on
  • The day to night progression, built as its own state machine
  • Audio occlusion for more immersive sound, plus ambient noise I recorded and sound effects I edited myself
  • The dialogue system, menus, settings, pause, HUD, and a flashlight with a battery meter

    • Development Insights

    This project has been in the works for over a year, and it's been a labor of love. I wanted to make a horror game that was actually scary, but more than that, I wanted it to capture what it feels like to be a kid who's afraid of the dark. It's taught me a ton, not just about building systems, but about sticking with one project long enough to see it through and going back to rebuild parts when they weren't good enough. Insomnia is the most challenging thing I've made, and easily the most rewarding.
