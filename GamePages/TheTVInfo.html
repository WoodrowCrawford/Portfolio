Project Overview

Play as a boy whose one goal is to watch TV. But the more you play, the more you'll find that things aren't what they seem. The TV is a short horror game built around a daily routine that slowly unravels the longer you keep watching, designed to creep up on you with each passing day.

Key Features

  • An old-school Game Boy visual style
  • A repeating daily loop where the story shifts a little each day
  • Branching events that change what's on the TV depending on the day
  • Runs as a real Game Boy ROM, on emulators and actual hardware like the 3DS

Development Details

  • Role: Solo Developer
  • Engine: GB Studio
  • Language: GBScript
  • Status: Completed
  • Platforms: Gameboy emulators and hardware, Web Browsers

What I worked on

I made this solo to teach myself GB Studio, and most of the actual work was building the game's logic out of GB Studio's event blocks rather than just writing dialogue. The main pieces:

  • A day system that tracks which day it is and what the player's supposed to be doing
  • Logic that checks the day and task to decide what shows up on the TV
  • Five scenes (bedroom, bathroom, living room, the dreams, and inside the TV) wired together into the daily loop

Development Insights

I made this purely to teach myself GB Studio, and it turned into a fun challenge. Building everything out of event blocks instead of code was a totally different way of thinking. I even exported it as a real Game Boy ROM and got it running on my 3DS, which was honestly the coolest part, seeing something I made boot up on actual hardware. I learned a lot about how much you can pull off inside a really restricted old-school toolset.
