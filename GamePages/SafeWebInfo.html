Safe Web
A Deep Dive

A visual novel created in Unity for a horror game jam. Includes a virtual computer interaction system.

Project Overview

Safe Web is a visual novel game I helped develop for a horror game jam. The game features a virtual computer interaction system where players can navigate through files, go on the web, and uncover the story. The game combines elements of horror and mystery to create an engaging experience.

Key Features

  • Virtual computer interaction
  • Voice acting
  • Retro aesthetics
  • Engaging horror narrative

Development Details

  • Engine: Unity
  • Language: C#
  • Status: Completed
  • Platforms: Windows, Mac

Development Insights

Working on Safe Web was a unique experience as it involved creating a virtual computer system within the game. My role in this project was the main programmer, and I was responsible for implementing the core mechanics of the virtual computer. This required careful planning and implementation to ensure that players could interact with the system seamlessly. The voice acting added an extra layer of immersion, enhancing the overall atmosphere of the game. Collaborating with a team to bring this project to life was both challenging and rewarding.
